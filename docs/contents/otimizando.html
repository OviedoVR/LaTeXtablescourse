Otimizando a criação de Tabelas em LaTeX

Autor

Vinícius Oviedo

Ferramentas

Certamente, existem ferramentas que nos auxiliam a otimizar a criação de tabelas em LaTeX. Alguns exemplos são:

  1. LaTeX tables generator
  2. Table Convert
  3. Plugins do MS Excel (e.g., Excel2LaTeX)
  4. Python (o pacote pandas tem suporte para TeX)

Essas ferramentas são incríveis, porém, elas não fazem todo o trabalho. Normalmente, dependendo da complexidade da tabela, é preciso implementar alguns ajustes. Por esse motivo, é interessante entender o básico sobre a estrutura de tabelas em LaTeX. Segue o processo que utilizei/utilizo ao longo tempo trabalhando com tabelas em LaTeX:

Exemplo com Python

Para esse exemplo, vamos utilizar o Google Colab. Vamos partir de um arquivo .csv e também de uma tabela da web (em html). Ao final, teremos duas tabelas em LaTeX de maneira prática e rápida.

Podemos utilizar a biblioteca Pandas para ler essa tabela:

# biblioteca:
import pandas as pd

# lendo a tabela:
dados = pd.read_csv("ordens.csv", sep=";")

2. Tabela html (da web)

wikipedia = pd.read_html("https://pt.wikipedia.org/wiki/Lista_de_unidades_federativas_do_Brasil_por_popula%C3%A7%C3%A3o")
wikipedia = wikipedia[2]
wikipedia = wikipedia[['Região', 'População (Estimativa 2024)']]
wikipedia
Região População (Estimativa 2024)
0 Região Sudeste 88.617.693
1 Região Nordeste 57.112.096
2 Região Sul 31.113.021
3 Região Norte 18.669.345
4 Região Centro-Oeste 17.071.595 
wikipedia.to_latex(index=False, caption='População BR por região', label='tab:populacao')
